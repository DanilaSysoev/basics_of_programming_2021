Задача 5. Иммитация работы двоичных деревьев поиска
===============================================================================
0. В рабочем репозитории в директрии 2sem/hw создайте директорию task_5.
1. Выполните задание в соответствии со своим вариантом.
2. Составьте протокол выполнения работы согласно следующим правилам:
   - протокол должен находиться в директории 2sem/hw/task_5;
   - для каждой задачи в протоколе должен быть блок строк, начинающийся
     с заголовка <X>, где <X> - название задания;
   - следом за заголовком должен идти блок строк описания результата решения
     задачи; строка описания - это описание состояния узла дерева;
     формат строки описания следующий:
     <узел> <левый> <правый> [<цвет>]
     где
     <узел> - ключ некоторого узла дерева
     <левый> - ключ его левогл потомка (x в случае если потомка нет)
     <правый> - ключ его правого потомка (x в случае если потомка нет)
     [<цвет>] - цвет узла (только для RB-дерева, r - красный, b - черный).
     Для RB-дерева nil узлы не указываем;
   - в протоколе разрешены пустые строки
3. Запуште изменения в облако.
