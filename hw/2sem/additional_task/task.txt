Дополнительное задание
===============================================================================
Для выполнения заданий можно использовать один из следующих языков программиро-
вания: Java, C++, C#, Python.
Для реализации интерфейса пользователя можно использовать какой-либо GUI фрейм-
ворк либо консольный интерфейс в стиле интерактивного меню.
Задачу можно выбрать из списка, либо предложить свою тему работы.


Задачи для одного
-------------------------------------------------------------------------------
1. Создание модификации или расширения к одной из игр:
DwarfFortress - через файлы ресурсов и скриптов.
NetHack - через анализ и модификацию исходного кода и ресурсов.
Можно предложить свою игру для модификации.

2. Создание полезной утилиты для упрощения игры в играх со сложным геймплеем.
(DwarfFortress, NetHack). Допускаются утилиты, ломающие баланс (чит-утилиты)


Задачи для команд
-------------------------------------------------------------------------------
1. Движок для текстового квеста.
Движок должен позволять описывать структуру текстового квеста в виде файлов.
Плюсом будет возможность добавления скриптов и возможность исполнения их при 
выборе соответствующих опций. 
Начальная информация по текстовым квестам:
https://ru.wikipedia.org/wiki/Interactive_fiction
К движку нужно подготовить документацию.

2. Игра в стиле rouguelike. Из базовых канонов должны быть соблюдены процедур-
ная генерация мира, перманентная смерть, ascii графика, пошаговость. 
Сеттинг может быть любым (космосим, фентези и т.д.)
Начальная инфа по roguelike
https://ru.wikipedia.org/wiki/Roguelike
